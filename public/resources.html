Resources

HELLO. here are some more lists of things that you can check out if you'd like! mostly things i thought would be helpful/have used in the past.

Web Dev

Self-Study

Other than the classics like W3schools & Geeks4geeks, here are some sites that helped me immensely while making this website.

  • sadgrl.online: my intro to the neocities scene & a great resource for beginners! plenty of easy guides & tools here.
  • petrapixel: tutorials, tools & templates, and a very handy coding knowledge checklist
  • A Guide to Flexbox: extremely comprehensive guide to flexbox. highly recommend going through the tutorial & learning as you go!
  • Eloquent Javascript: an online book on JS that i haven't actually gotten through in its entirety, but may be helpful for beginner coders!
  • Writing Alt Text for Accessibility: Guidelines & Examples

Assets

  • Artwork's Graphic Collection (flash/moving gifs warning!)
  • Bonnibel's Graphic Collection
  • Coco's Pixel Safari: blinkies, buttons, pixels, etc!
  • minnow's graphics carrd: blinkies, gifs, dividers and more
  • sadgrlonline's tiled backgrounds archive
  • Assorted text graphic makers
  • Dafont: free fonts
  • Pokemon Crystal Sprite Archive!

Tools

  • WAVE Accessibility Checker: website that scans your website and tells you how accessible your site is and how to fix it if it's not
  • Color Hex: hex code color picker & palettes
Art/Design

Learning

I will be real I mostly learned by doing, but any resources I can find for these that I've saved over the years, I'll put here!

  • Sinix: Holy grail of Youtube's art tutorials
  • a HUGE collection of drawing reference sites, which include a very large number of museums/institutions
  • A Compilation of Youtube Blender Tutorials by Mpekamitzii
  • The Creator's Guide to Comics Devices: library of comic/visual narrative devices
  • A brief guide to drawing East Asian faces by Kat Tsai
  • a few tips on drawing fat people
  • a helpful starting point for drawing avian wings

Tools/Freeware/etc.

  • Krita: free art software!
  • Blender: 3d modeling & animating program
  • Affinity: alternative to Adobe suite made by Canva
  • Gumroad: great place to find free/paid brushes, assets, look books, etc. among other things
  • Photopea: another Adobe alternative
  • Ataro head: rotatable 3d head!
Game Dev

Engines

  • Godot: open source game engine that uses a Python-based programming language or C#
  • Ren'Py: an open source visual novel/game engine with extensive documentation & help forums; great for beginners
  • Windowpane: simple web game engine for games, visual novels, interactive fiction, etc.
  • Bitsy: a very tiny game engine for very tiny games
  • Twine: another interactive fiction maker for people with less coding experience

Tools/Assets

  • Freesound: ...free sounds...
  • Libresprite: open-source & free Aseprite alternative
  • itch.io: for games, assets, & inspiration

Self-Study

  • The Book of Shaders: An extensive online guide to fragment shaders for use on any platform that supports GLSL
Reading

Online Libraries

  • Internet Archive
  • Libby: Library app you can access with multiple library cards
  • ^ supplemental: you can check if your local library is connected to Libby here!
  • Books Unbanned: US only, but provides digital library cards to youth aged 13-26
  • QLL: also US only :( provides digital library cards to anyone in the country
